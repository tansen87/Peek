# Peek

Peek 是一款高效的大文件文本查看器,基于Tauri构建,支持快速打开和浏览超大文本文件

## 功能特性

### 核心功能
- **大文件支持**: 采用内存映射技术,可快速打开和浏览 GB 级别的超大文本文件
- **多标签浏览**: 支持同时打开多个文件,通过标签页快速切换
- **编码检测**: 自动检测文件编码,支持 UTF-8、UTF-16、GBK、GB18030、BIG5 等多种编码

### 搜索与替换
- **快速搜索**: 支持在文件中快速查找文本
- **正则表达式**: 支持正则表达式模式进行高级搜索
- **区分大小写**: 可选择是否区分大小写进行搜索
- **文本替换**: 支持查找并替换文件中的文本内容

### 文件导航
- **行号跳转**: 快速跳转到指定行号
- **搜索结果高亮**: 在文件中突出显示所有匹配项
- **当前行指示**: 清晰显示当前光标所在行

### 用户界面
- **深色/浅色模式**: 支持主题切换
- **系统托盘**: 可最小化到系统托盘,方便快速访问
- **流畅滚动**: 即使大文件也能保持流畅的滚动体验

## 使用方法

1. 启动 Peek 应用
2. 点击工具栏上的文件夹图标选择要打开的文件
3. 也可以直接将文件拖拽到 Peek 窗口中打开

### 文件操作

| 操作 | 说明 |
|------|------|
| 打开文件 | 点击工具栏图标或通过命令行传递文件路径 |
| 切换标签 | 点击标签页或使用快捷键 |
| 关闭标签 | 点击标签页上的关闭按钮 |
| 搜索文本 | 点击工具栏搜索图标或使用快捷键 |
| 跳转行号 | 点击工具栏的行号图标,输入目标行号 |
| 切换编码 | 点击工具栏编码选择器,选择目标编码 |

### 快捷操作

- **打开**: `Ctrl + O`
- **搜索**: `Ctrl + F`
- **替换**: `Ctrl + H`
- **跳转**: `Ctrl + G`
- **关闭**: `Ctrl + W`
